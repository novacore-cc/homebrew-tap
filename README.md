# Novacore Tap

## How do I install these casks?

`brew install --cask novacore-cc/tap/<cask>`

Or `brew tap novacore-cc/tap` and then `brew install --cask <cask>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "novacore-cc/tap"
cask "<cask>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
